# User Guide

## Getting Started
To start using the application, follow the installation instructions provided in the README.

## Using the Vulnerability Scanner
1. Navigate to the Scanner page.
2. Enter the URL you wish to scan.
3. Click on the "Scan" button.
4. View the results displayed.

## Using the Malware Analysis Sandbox
1. Navigate to the Malware Analysis page.
2. Upload the suspicious file.
3. Click on "Analyze".
4. View the results displayed.

## Using the Data Breach Checker
1. Navigate to the Data Breach Checker page.
2. Enter your email address.
3. Click on "Check".
4. View the results displayed.
